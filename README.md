# AEME - Avaliação Estadual de Melhoria Educacional

O **AEME** é um projeto desenvolvido como proposta de um "ENEM 2.0". Ele visa substituir o modelo tradicional de provas padronizadas por um sistema de avaliação personalizado, focado nas reais habilidades e talentos de cada participante.

Este projeto foi desenvolvido para a disciplina de Desenvolvimento de Sistemas do **SENAI** e tem como público-alvo de apresentação a **Secretaria de Desenvolvimento Econômico do Estado de São Paulo** e o **Instituto de Pesquisas Tecnológicas (IPT)**.

## 🎯 Objetivo do Projeto
Dar oportunidade e visibilidade para talentos frequentemente invisibilizados pelo sistema escolar brasileiro. Com a nota obtida no AEME, o participante pode conquistar uma bolsa de estudos em uma faculdade da sua respectiva área.

## 💡 Como Funciona?
- Ao invés de uma prova de múltiplas disciplinas (Matemática, Português, etc.), o candidato desenvolve um **projeto prático** relacionado à área que ele mais domina (ex: uma obra de arte, um software, etc.).
- A nota final é composta pela média entre a avaliação de um **corretor profissional** e a análise de uma **Inteligência Artificial** configurada com os critérios específicos de cada área de atuação.
